# IPorts Interface

[? Back to I/O API](README.md)

Defines the contract for the Z80 I/O port system.

## Key Methods
- `byte Read(byte port)`: Reads from a port.
- `void Write(byte port, byte value)`: Writes to a port.
- `void Clear()`: Clears all port states.
